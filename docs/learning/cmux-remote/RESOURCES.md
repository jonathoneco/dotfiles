# cmux remote-tmux resources

## Knowledge

- [cmux SSH documentation](https://cmux.com/docs/ssh)
  Current cmux behavior for SSH configuration, reconnection, remote browser routing, file transfer, notifications, and the relay daemon.
- [cmux CLI help](https://github.com/manaflow-ai/cmux)
  Source and release context for the nightly `ssh-tmux` beta.
- [tmux control mode](https://github.com/tmux/tmux/wiki/Control-Mode)
  Primary explanation of the protocol cmux uses to render a tmux server through a native interface.

## Wisdom (Communities)
