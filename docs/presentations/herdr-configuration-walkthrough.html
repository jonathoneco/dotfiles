04 · Workspace navigation

Use project navigation instead of hunting for a terminal.

Find and switch

Prefix f opens Herdr’s navigator. Prefix Shift-f opens the workspace picker.

Cycle and jump

Prefix Shift-←/→ cycles workspaces. Prefix Shift-1…9 jumps directly by index.

This is the key change from a pure tmux port: the terminal layout is no longer the only unit of navigation. Projects become first-class.

05 · Project entry and worktrees

Sessionizer finds projects; Herdr owns worktree lifecycle.

Project picker

Prefix Ctrl-s opens the installed Sessionizer plugin’s fuzzy project picker.

New worktree

Prefix Shift-g creates a native, current-workspace-aware worktree.

Open worktree

Prefix Shift-o opens an existing worktree as a workspace.

Why native?

Herdr tracks the worktree’s workspace and distinguishes closing it from deleting the checkout.

06 · Focused terminal workflow

Temporary tools become panes; output can become an editor buffer.

Scrollback

Prefix u opens the focused pane’s scrollback in $EDITOR.

Tool panes

Prefix g lazygit, Prefix e ranger, Prefix Shift-d lazydocker, and more.

Scratch space

Prefix w opens a scratch shell; notes have persistent and ephemeral bindings.

Herdr does not promote a completed temporary pane into a tab. The pane stays alive only while its command runs.
07 · Agents

Agents are visible, resumable project state—not just processes in panes.

Launch

Prefix Alt-j Claude, Prefix Alt-k Codex, Prefix Alt-l Pi, and Prefix Alt-; agent.

Observe

Pane borders show agent labels; the agent panel is sorted by priority, surfacing blocked or completed work.

Restore

Claude, Codex, and Pi integrations are installed. resume_agents_on_restore = true lets Herdr restore supported conversations after restart.

Important limit

Launching an agent key opens a new pane each time. Herdr has no native equivalent to the old cached tmux agent window.

08 · Attention without noise

Let background work reach you, without turning the terminal into a bell.

[ui.toast]
delivery = "system"
delay_seconds = 10

[ui.sound]
enabled = false

A notification is sent only after attention persists for 10 seconds; sound remains off. Prefix o opens the notification’s target.

09 · Remote-ready from your Mac

Remote sessions keep local controls—and can receive pasted images.

Remote attach

Use herdr --remote for supported Linux or macOS hosts while retaining local keybindings.

Image paste

The explicit disablement was removed, restoring Herdr’s Ctrl-v remote image-paste default.

Link handling

Use Herdr’s Ctrl-click link opener or the terminal’s usual URL/file handling.

Scrollback budget

10 MiB per pane, expressed as bytes rather than tmux lines.

10 · Intentional boundaries

Not every tmux behavior should be simulated.
