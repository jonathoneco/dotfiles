# adr-0001-<decision>.md

Use for hard-to-reverse or easily re-litigated decisions.

Include:

- context
- decision
- rejected alternatives
- consequences
- reversibility

The ADR should record why the chosen shape won, not restate the whole design spec.

## Template

```md
# ADR 0001: <Decision>

Status: <proposed | accepted | superseded>

## Context

...

## Decision

...

## Rejected alternatives

...

## Consequences

...

## Reversibility

...
```
